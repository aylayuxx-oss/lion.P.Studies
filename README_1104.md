# [백엔드] Python 첫걸음 시작하기

## [3] Git설치와 Github가입
### 1. 로컬 저장소와 GitHub 원격 저장소를 연동 완료했다.
### 2. 수정한 파일을 git add로 스테이징하고 git commit으로 변경 내용을 기록했다.
### 3. git push 명령으로 원격 저장소에 업로드했다.
### 4. git clone으로 원격 저장소를 복제해 병렬로 작업하는 방법을 실습했다.
### 5. git checkout으로 브랜치를 전환해 동시에 여러 작업을 관리했다.
### 6. Git과 GitHub를 활용해 협업 시 코드 이력 관리와 충돌 방지 방법을 이해했다.
