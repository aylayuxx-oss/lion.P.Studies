# 명령어 한줄로 숏폼 영상 만들기

## 1. AI 이미지 생성과 AI 영상 생성
### (1) AI 이미지 생성 → 텍스트 기반 명령어로 이미지를 생성하는 방법 실습.
### (2) AI 영상 생성 → 이미지 또는 프롬프트로 짧은 영상 생성 워크플로우 학습.

## 2. AI 영상편집 도구의 개요 및 AI 편집 기능 활용
### (1) AI 영상편집 도구의 개요 및 AI 편집 기능 활용 → 주요 AI 편집 툴(Vrew, Pictory 등)의 기능 비교 및 자동화 활용법.
### (2) 다양한 영상편집 도구 개요 → 무료·유료 도구별 특징과 실제 사용 시 고려 요소 정리.

## 3. 음성 인식 자막 생성 및 싱크 조정
### (1) Vrew 자막 생성 → 음성 인식 기반 자동 자막 생성 과정 이해 및 실습.
### (2) Vrew 자막 싱크 조정 → 자동 자막의 싱크 문제를 수동 보정하는 실습 중심.

## 4. AI 영상 활용 사례 및 30초 영상 콘텐츠 만들기
### (1) AI 영상 활용 사례 → 기업·마케팅 분야에서 AI 영상 활용한 실제 성공사례 공유.
### (2) 30초 영상 만들기 → AI 도구로 짧은 숏폼 콘텐츠를 기획·제작하는 종합 실습.
